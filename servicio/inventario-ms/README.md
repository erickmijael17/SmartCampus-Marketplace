# Microservicio Inventario

Microservicio para controlar stock disponible por producto en CampusMarket.

## Responsabilidad
- Registrar inventario por producto.
- Consultar inventario por producto.
- Actualizar stock disponible y reservado.

## Endpoints
- POST /api/v1/inventarios
- GET /api/v1/inventarios
- GET /api/v1/inventarios/{id}
- GET /api/v1/inventarios/producto/{idProducto}
- PUT /api/v1/inventarios/{id}
- DELETE /api/v1/inventarios/{id}
